# MongoDB + Python Web Catalog (CST8276)

Group project for **Advanced Database Topics**.

This repo currently contains the **skeleton** created by Mehrad:

- Project structure
- MongoDB connection helper (`db.py`)
- Collection validator script (`validators.js`)
- Seed script with sample products & reviews (`seed.py`)
- FastAPI app skeleton (`app.py`) with stub routes
- `requirements.txt` and `.env.example`

## Setup

1. Create and activate virtual environment (optional but recommended).

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
